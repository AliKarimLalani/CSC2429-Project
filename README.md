# CSC2429 Project

This repository contains my write-up and presentation slides for the graduate course CSC2429H — Topics in the Theory of Computation: Algebraic Complexity at the University of Toronto, taught by Professor Shubhangi Saraf.  The work focuses on Ran Raz's breakthrough 2010 paper, "Tensor-rank and lower bounds for arithmetic formulas". It explores the core result showing that strong enough lower bounds for tensor rank can yield super-polynomial lower bounds for general arithmetic formulas.
